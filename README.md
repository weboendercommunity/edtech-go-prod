# Edtech

API collection for Edtech application.

## Tech Stack

**Server:** Go


## Run Locally

Clone the project

```bash
  git clone https://github.com/shevaathalla/edtech-go.git
```

Go to the project directory

```bash
  cd edtech-go
```

Install dependencies

```bash
  go mod download
```

Start the server

```bash
  go run cmd/api/main.go
```
